## Списки

Чтобы добавить ненумерованные списки, нужно пункты выделить звёздочкой (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Нумерованные - просто цифры с точкой:
1. Вот один
2. Вот второй
